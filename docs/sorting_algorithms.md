# Алгоритмы сортировки

## Описание

Этот проект посвящен реализации и сравнению различных алгоритмов сортировки. Все алгоритмы применяются к строкам матрицы, каждая строка сортируется независимо.

## Реализованные алгоритмы

### 1. Сортировка пузырьком (Bubble Sort)
- **Сложность времени**: O(n²)
- **Сложность памяти**: O(1)
- **Описание**: Простой алгоритм, который многократно проходит по списку, сравнивает соседние элементы и меняет их местами, если они находятся в неправильном порядке.

### 2. Сортировка вставками (Insertion Sort)
- **Сложность времени**: O(n²)
- **Сложность памяти**: O(1)
- **Описание**: Алгоритм строит отсортированную последовательность по одному элементу, вставляя каждый новый элемент в правильную позицию среди уже отсортированных элементов.

### 3. Быстрая сортировка (Quick Sort)
- **Сложность времени**: O(n log n) в среднем, O(n²) в худшем случае
- **Сложность памяти**: O(log n)
- **Описание**: Эффективный алгоритм, использующий стратегию "разделяй и властвуй". Выбирает опорный элемент и разделяет массив на части.

### 4. Сортировка выбором (Selection Sort)
- **Сложность времени**: O(n²)
- **Сложность памяти**: O(1)
- **Описание**: Алгоритм находит минимальный элемент в неотсортированной части массива и помещает его в начало.

### 5. Сортировка Шелла (Shell Sort)
- **Сложность времени**: O(n^(3/2)) в среднем случае
- **Сложность памяти**: O(1)
- **Описание**: Улучшенная версия сортировки вставками, использующая убывающие интервалы для сравнения элементов.

### 6. Турнирная сортировка (Tournament Sort)
- **Сложность времени**: O(n²)
- **Сложность памяти**: O(n)
- **Описание**: Алгоритм находит минимальный элемент и последовательно добавляет его в отсортированный список.

## Структура файлов

- `main.py` - модуль для генерации случайных матриц
- `bubble_sort.py` - реализация сортировки пузырьком
- `insertion_sort.py` - реализация сортировки вставками
- `quick_sort.py` - реализация быстрой сортировки
- `selection_sort.py` - реализация сортировки выбором
- `shell_sort.py` - реализация сортировки Шелла
- `tournament_sort.py` - реализация турнирной сортировки
- `sorting_algorithms.ipynb` - Jupyter notebook с экспериментами и анализом алгоритмов сортировки

## Использование

Каждый файл сортировки можно запустить независимо:

```bash
python bubble_sort.py
```

При запуске программа запросит параметры матрицы:
- Количество строк (по умолчанию 50)
- Количество столбцов (по умолчанию 50)
- Минимальное значение элементов (по умолчанию -250)
- Максимальное значение элементов (по умолчанию 1005)

Программа выведет исходную матрицу, отсортированную матрицу и время выполнения алгоритма.

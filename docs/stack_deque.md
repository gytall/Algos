# Структуры данных: Стек и Дек

## Описание

Этот проект посвящен реализации и применению двух основных структур данных: стека (stack) и дека (deque - двусторонняя очередь). Разработаны программы для обработки данных из текстовых файлов с использованием этих структур.

## Реализованные структуры данных

### Стек (Stack)
Стек - это структура данных, работающая по принципу LIFO (Last In, First Out - последний пришел, первый ушел).

**Операции:**
- Инициализация
- Проверка на пустоту
- Добавление нового элемента в начало (push)
- Извлечение элемента из начала (pop)

### Дек (Deque - Двусторонняя очередь)
Дек - это структура данных, позволяющая добавлять и удалять элементы с обоих концов.

**Операции:**
- Инициализация
- Проверка на пустоту
- Добавление нового элемента в начало
- Добавление нового элемента в конец
- Извлечение элемента из начала
- Извлечение элемента из конца

## Задания

### Задание 1: Сортировка книг
Отсортировать строки файла, содержащие названия книг, в алфавитном порядке с использованием двух деков.

**Алгоритм:**
1. Чтение названий книг из файла в первый дек
2. Поиск минимального элемента и перемещение его во второй дек
3. Повторение до опустошения первого дека

### Задание 2: Расшифровка сообщения
Дек содержит последовательность символов для шифровки сообщений. Расшифровать текст, зная, что при шифровке каждый символ заменялся следующим за ним в деке по часовой стрелке через один.

**Алгоритм:**
- Использование дека ASCII символов
- Вращение дека для расшифровки

### Задание 3: Ханойские башни
Реализовать алгоритм переноса n дисков со стержня А на стержень С, используя три стека вместо стержней.

**Правила:**
- На каждом шаге переносится только один диск
- Диск нельзя помещать на диск меньшего размера
- Для промежуточного хранения используется стержень В

### Задание 4: Проверка баланса круглых скобок
За один просмотр файла проверить баланс круглых скобок в тексте программы, используя стек.

**Алгоритм:**
- При встрече '(' - добавляем в стек
- При встрече ')' - извлекаем из стека
- Если стек пуст при встрече ')' или не пуст в конце - скобки несбалансированы

### Задание 5: Проверка баланса квадратных скобок
За один просмотр файла проверить баланс квадратных скобок в тексте программы, используя дек.

### Задание 6: Группировка символов
Используя стек, за один просмотр файла напечатать сначала все цифры, затем все буквы, и, наконец, все остальные символы, сохраняя исходный порядок в каждой группе.

### Задание 7: Группировка чисел
Используя дек, за один просмотр файла напечатать сначала все отрицательные числа, затем все положительные числа, сохраняя исходный порядок в каждой группе.

### Задание 8: Обращение строк
Используя стек, сформировать новый текстовый файл, содержащий строки исходного файла, записанные в обратном порядке.

## Структура файлов

- `stack_deque.ipynb` - Jupyter notebook со всеми реализациями структур данных и примеров их использования
- `books.txt` - файл с названиями книг
- `encrypted.txt` - зашифрованное сообщение
- `hanoi_disks.txt` - файл с количеством дисков для задачи о Ханойских башнях
- `program_brackets.txt` - программа для проверки круглых скобок
- `program_square_brackets.txt` - программа для проверки квадратных скобок
- `characters.txt` - файл с символами для группировки
- `numbers.txt` - файл с числами для группировки
- `lines.txt` - файл со строками для обращения

## Использование

Откройте `stack_deque.ipynb` в Jupyter Notebook и выполните ячейки по порядку. Убедитесь, что все необходимые текстовые файлы находятся в той же директории.

## Зависимости

- `collections.deque` - для реализации дека
- Стандартная библиотека Python для работы с файлами
